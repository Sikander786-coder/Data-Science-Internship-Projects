# Personal Loan Acceptance Prediction

## Objective
Predict whether a customer will accept a personal loan offer based on demographic and financial data.

## Dataset
Bank Marketing Dataset containing features such as age, job, marital status, balance, and contact details.

## Steps Performed
- Data exploration and visualization
- Analysis of customer demographics
- Data encoding using one-hot encoding
- Model training using Logistic Regression and Decision Tree
- Model evaluation using accuracy and confusion matrix

## Key Insights
- Customers with higher account balances are more likely to accept loan offers.
- Certain job categories respond better to marketing campaigns.
- Middle-aged customers show higher engagement.

## Conclusion
Machine learning models can help banks target the right customers and improve marketing effectiveness.
