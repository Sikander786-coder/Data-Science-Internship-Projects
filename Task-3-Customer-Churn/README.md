# Customer Churn Prediction

## Objective
Identify customers who are likely to leave the bank using machine learning techniques.

## Dataset
Churn Modelling Dataset containing customer details such as age, balance, geography, and activity status.

## Steps Performed
- Data cleaning and removal of irrelevant columns
- Exploratory Data Analysis (EDA)
- One-hot encoding of categorical variables
- Model training using Random Forest
- Feature importance analysis

## Key Insights
- Age, balance, and activity status are key factors influencing churn.
- Older customers are more likely to leave the bank.
- Active members are less likely to churn.

## Conclusion
Understanding churn behavior helps banks develop strategies to retain customers and improve customer satisfaction.
