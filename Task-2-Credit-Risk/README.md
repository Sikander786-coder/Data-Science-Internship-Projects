# Credit Risk Prediction

## Objective
Predict whether a loan applicant is likely to default based on personal and financial information.

## Dataset
Loan Prediction Dataset containing features such as income, loan amount, education, and credit history.

## Steps Performed
- Data cleaning and handling missing values
- Exploratory Data Analysis (EDA)
- Feature encoding
- Model training using Logistic Regression and Decision Tree
- Model evaluation using accuracy and confusion matrix

## Key Insights
- Credit history is the most important factor influencing loan approval.
- Applicants with higher income are more likely to be approved.
- Logistic Regression provides more stable predictions compared to Decision Tree.

## Conclusion
Machine learning models can effectively assist financial institutions in assessing loan risk and improving decision-making.
