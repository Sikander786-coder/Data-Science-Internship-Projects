# Insurance Cost Prediction

## Objective
Predict medical insurance charges based on personal attributes such as age, BMI, and smoking status.

## Dataset
Medical Cost Personal Dataset containing features like age, BMI, number of children, smoking status, and charges.

## Steps Performed
- Data exploration and visualization
- Feature analysis using scatter plots and box plots
- Data encoding
- Model training using Linear Regression
- Model evaluation using MAE and RMSE

## Key Insights
- Smoking is the most significant factor affecting insurance charges.
- Age has a positive relationship with charges.
- BMI has moderate impact on charges.

## Conclusion
Regression models can be used to estimate insurance costs and support decision-making in the healthcare industry.
